# instagram-comment-analyzer
<h3>Como rodar o front-end:</h3>
<ul>
  <li>Instale o Node</li>
  <li>npm install</li>
  <li>npm start</li>
</ul>

<h3>Como rodar a API: </h3>
<ul>
  <li>Instali o SDK do .net(https://dotnet.microsoft.com/download)</li>
  <li>dotnet add package</li>
  <li>dotnet run</li>
</ul>
